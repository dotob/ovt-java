OVT Software - Java Installationsanleitung

 Vorab - falls noch nicht geschehen - legen Sie in Ihrer Datei "Eigene Dateien" einen neuen Ordner "Overtür" an, wo Sie alle für Ihre Zusammenarbeit mit Overtür anfallenden Programme und Dateien speichern werden.

  1. Öffnen Sie Ihren Ordner  "Eigene Dateien/Overtür"
  2. Danach hier http://java.com/de anklicken.
  3. Klicken Sie auf der Seite 1 der Java-Software unterhalb von "Download NOW!" auf Manuell herunterladen .
  4. Klicken Sie in der Spalte SCHRITT 1: DOWNLOAD neben  Windows (Offline-Installation) auf Download
  5. Klicken Sie im Dialogfeld Dateidownload auf Speichern .
  6. Wählen Sie im Dialogfeld Datei speichern unter als Speicherort für die Datei "jre-1_5_0_(Version-Nr.)-windows-i586-p.exe" den von Ihnen angelegten Ordner "Overtür".
  7. Klicken Sie auf Speichern .  Die Dauer wird Ihnen im Dialogfeld angezeigt. Falls diese Java-Software wider Erwarten bereits vorhanden ist, werden Sie gefragt: Möchten Sie sie ersetzen? Klicken Sie dann auf Ja .
  8. Im Dialogfeld Download beendet (Installation beendet) klicken Sie auf Schließen
  9. Schließen Sie die Java-Software und alle Anwendungen, darunter auch den Browser.
  10. Öffnen Sie nun wieder Ihren Ordner "Eigene Dateien/Overtür". Klicken Sie dort doppelt auf das Symbol für die gespeicherte Datei "jre-1_5_0_(Version-Nr.)-windows-i586-p.exe", um den Installationsvorgang zu starten. Sie werden im Dialog durch den Vorgang geführt.



 Das Installationsprogramm extrahiert die Dateien, die für die weiteren Schritte erforderlich sind. Dieser Vorgang, der weniger als eine Minute dauert, kann in einem Dialog verfolgt werden. Das Installationsprogramm blendet kurz einen Begrüßungsbildschirm mit dem Logo ein und zeigt dann den Lizenzvertrag an.



Lesen Sie die Lizenzvereinbarung durch, und lassen Sie den Punkt Typische Installation markiert. Um die Bedingungen zu akzeptieren, klicken Sie auf  Accept , um mit der Installation fortzufahren.

Nachdem Sie alle erforderlichen Angaben gemacht haben, können Sie den Fortschritt der Installation im Dialog verfolgen.


In einigen kurzzeitig eingeblendeten Dialogen werden die letzten Schritte des Installationsvorgangs bestätigt, und es wird eine Bestätigungsmeldung über den ordnungsgemäßen Abschluss der Installation angezeigt.  Klicken Sie hier auf Fertigstellen und schließen Sie damit die Installation ab.
