2. Schritt: RMK MF-Assistent installieren
Öffnen Sie Ihren Ordner "Eigene Dateien/Remarkom".
Laden Sie mit dem folgenden Link den RMK MF-Assistenten in das Verzeichnis "Eigene Dateien/Remarkom" herunter. Nach beendetem Download führen Sie die Datei rmkmfassistent.exe aus (Doppelklick). Folgen Sie den Anweisungen auf dem Bildschirm. Danach befindet sich auf Ihrem Desktop ein Symbol, mit dem Sie den MF-Assistenten starten können.
